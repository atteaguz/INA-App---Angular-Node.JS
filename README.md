# INA App - Sistema de Gestión

# Autor:
José Agustín Zelaya Solórzano

Sistema de gestión desarrollado como parte de las clases prácticas en el INA - Costa Rica.

## Tecnologías utilizadas

### Frontend
- Angular 21
- Angular Material
- TypeScript
- Signals

### Backend
- Node.js
- Express
- TypeORM
- MySQL
- JWT para autenticación

## Funcionalidades

- ✅ Autenticación de usuarios (login/logout)
- ✅ CRUD de Categorías
- ✅ CRUD de Usuarios
- ✅ Control de roles (admin/user/guest)
- ✅ Dashboard
- ✅ Diseño responsivo con Angular Material

## Instalación y configuración

### Requisitos previos
- Node.js (v18 o superior)
- MySQL
- Angular CLI

### Backend
```bash
cd API
npm install
npm run dev
